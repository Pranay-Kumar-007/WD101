5.6 Semantic Diff Engine

Instead of raw text diff, configs are parsed into structured network objects.

Object Type	Intelligence

VLAN	Added / Removed / Orphaned
Interface	Admin / Oper / VLAN / Trunk
ACL	Permit/Deny / Order Change
Route	Next-hop change
F5 VS	Pool member changes


Output Types

JSON diff

Engineering summary

Raw text diff

6. End-to-End Workflow Execution


---

6.1 Pre-Change Phase

1. Engineer submits change


2. Turbo validates:

IP reachability

Credentials



3. Devices executed in parallel


4. Snapshots labeled as:



Device_PRE_YYYYMMDD_HHMMSS.txt


---

6.2 Change Execution (External)

Turbo pauses

Engineer applies change

Turbo waits for POST trigger



---

6.3 Post-Change Phase

Same execution logic as PRE

Files labeled:


Device_POST_YYYYMMDD_HHMMSS.txt


---

6.4 Diff & Reporting Phase

PRE vs POST mapped

Semantic analysis performed

Risk engine triggered

Reports generated
